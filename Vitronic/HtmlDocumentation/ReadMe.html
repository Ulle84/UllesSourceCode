Bitte JavaScript aktivieren, um den vollen Dokumentationsumfang nutzen zu können!
Wenn JavaScript aktiviert ist:
  • wird das Inhalts-, Abbildungs- und Tabellenverzeichnis automatisch generiert
  • werden Verlinkungen zu Kapiteln, Tabellen und Abbildungen automatisch generiert
  • werden Tabellen- und Bildunterschriften automatisch generiert
  • werden in den Tabellen Tooltips erzeugt
  • wird die Dokument-Überschrift automatisch gesetzt

Anwendungsbereich

Das Paket ViAcquiDocumentation kann verwendet werden, um HTML-Dokumentation mit CSS-Stylesheets einheitlich zu gestalten und mit JavaScript mehr Funktionalität zu erzeugen.

Verwendung des CSS-Stylesheets

Das CSS-Stylesheet kann wie folgt eingebunden werden:
<link rel="stylesheet" type="text/css" href="style.css" />
Damit wird das Dokument einheitlich und besser leserlich gestaltet.

Aktivierung der JavaScript Funktionalität

Wenn JavaScript im Browser aktiviert ist, kann ein JavaScript wie folgt eingebunden werden:
<script type="text/javascript" src="script.js"></script>
Wenn im body-Tag die Funktion init() aufgerufen wird, z. B.
<body onload="init()">
dann
  • wird das Inhalts-, Abbildungs- und Tabellenverzeichnis automatisch generiert siehe , und
  • werden Verlinkungen zu Kapiteln, Tabellen und Abbildungen automatisch generiert siehe
  • werden Tabellen- und Bildunterschriften automatisch generiert, siehe und
  • werden in den Tabellen Tooltips erzeugt siehe

Dokumentüberschrift

Wenn im head Bereich des HTML Codes ein title Tag vorhanden ist, z. B.:
<head>
  <title>Titel des Dokumentes</title>
  ...
</head>
so wird am Anfang des Dokumentes (vor dem Inhaltsverzeichnis) eine Dokumentüberschrift mit dem Inhalt des title Tags erstellt.

Automatische Erstellung von Verzeichnissen

Es werden automatisch Inhalts-, Abbildungs- und Tabellenverzeichnisse erstellt.

Inhaltsverzeichnisses

Wenn man die Überschriften mit den HTML-Tags h1, h2, h3, h4, h5 und h6 auszeichnet, wird automatisch ein hierarchisch Inhaltsverzeichnis erstellt. Dafür muss es mindestens ein h1 Tag geben.
Die Kapitel und Unterkapitel werden automatisch durchnummeriert - man braucht keine Zahlen voranzustellen.

Abbildungsverzeichnisses

Wenn man Bilder im Dokument mit dem img Tag eingebunden hat, wird automatisch ein Abbildungsverzeichnis erzeugt.

Tabellenverzeichnisses

Wenn man Tabellen im Dokument mit dem table Tag eingebunden hat, wird automatisch ein Tabellenverzeichnisses erzeugt.

Automatische Erstellung von Unterschriften

Es werden automatisch Abbildungs- und Tabellenunterschriften generiert.

Abbildungs-Unterschriften

Wenn man Bilder im Dokument mit dem img Tag eingebunden hat und das title Attribut beschrieben hat, dann wird eine Bildunterschrift mit dem Inhalt des title Attributes automatisch generiert.

<img src="images/vitronic_logo.jpg" title="Beispielbild"/>

Tabellen-Unterschriften

Wenn man Tabellen im Dokument mit dem table Tag eingebunden hat und das title Attribut beschrieben hat, dann wird eine Tabellenunterschrift mit dem Inhalt des title Attributes automatisch generiert.
Hier ein Beispiel:
<table title="Bespieltabelle">
  ...
</table>
sieht so aus:
Überschrift Spalte 1 Überschrift Spalte 2
Zeile 1, Spalte 1 Zeile 1, Spalte 2
Zeile 2, Spalte 1 Zeile 2, Spalte 2

Verlinkungen innerhalb des Dokumentes

Wenn man Kapitel, Abbildungen oder Tabellen verlinken möchte, kann man im zu verlinkenden Element eine ID vergeben, z. B.
<table id="tabPraefixe">
  ...
</table>
Die ID muss eindeutig sein. Es bietet sich an folgende Präfixe zu verwenden:
Element Präfix
Kapitel chap
Bild img
Tabelle tab
Wenn man innerhalb des Textes (evt. an mehreren Stellen) auf das Element mit der festgelegten ID referenzieren möchte, dann kann man das wie folgt tun:
<span class="imgBeispiel">Abbildung 1</span>
Das sieht so aus:
Zu beachten ist, dass der Text innerhalb des span Tags komplett ersetzt wird, durch Kapitel x, Abbildung x oder Tabelle x. Sollte JavaScript nicht aktiviert sein, bleibt der Inhalt des span Tags erhalten und macht gegebenfalls keinen Sinn.

Tooltips in Tabellen

In allen Zellen der Tabellen werden automatisch Tooltips (Inhalt: Spaltenüberschrift, wenn vorhanden) gesetzt. Dies kann bei großen Tabellen mit vielen Spalten hilfreich sein, wenn die Spaltenüberschrift bereits nicht mehr im Sichtfeld ist.
